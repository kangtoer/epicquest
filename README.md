# 🗿 NUSANTARA EPIC QUEST
### Sistem Eksplorasi Arkeologi Fiksi v3.1
**Kapsul Waktu Nusantara** — Media Pembelajaran IPS Interaktif untuk Generasi Digital

---

## 📖 Deskripsi

**Nusantara Epic Quest** adalah platform pembelajaran berbasis HTML5 yang menggabungkan narasi sejarah, sosiologi, dan ekonomi Indonesia dalam pengalaman gaming interaktif. Dirancang khusus untuk mendukung **Profil Pelajar Pancasila**, aplikasi ini mengajak siswa dalam perjalanan lintas waktu dari Era Pra-Aksara hingga Indonesia Emas 2045.

Melalui dialog dengan tokoh bersejarah, quiz berbasis kasus nyata, dan mekanisme gameplay yang engaging, siswa mengembangkan:
- ✅ **Nalar Kritis** — Analisis fenomena sosial-ekonomi kontemporer
- ✅ **Kemandirian** — Literasi finansial dan kesadaran ekonomi
- ✅ **Berkebinekaan Global** — Pemahaman toleransi dan keberagaman
- ✅ **Gotong Royong** — Kesadaran persatuan dan kolaborasi

---

## 🎮 Fitur Utama

### 1. **Gameplay Berbasis Eksplorasi**
- Peta vertikal interaktif 4500px (Era Pra-Aksara → Indonesia Emas 2045)
- Kontrol WASD + Mouse atau Virtual Joystick (mobile-friendly)
- Sistem reward XP untuk mendorong engagement
- Partikel visual dan animasi dinamis

### 2. **Narasi Multibahasa**
Dialog karakter dalam **3 bahasa**:
- 🇮🇩 **Indonesian** — Bahasa utama formal
- 🇮🇩 **Javanese (Jawa Kuno)** — Pelestarian budaya lokal
- 🇬🇧 **English** — Aksesibilitas internasional

### 3. **Quiz Berbasis Kasus Nyata**
Setiap era memiliki pertanyaan yang terhubung dengan:
- **Data empiris** (BPS, BNPB, OJK)
- **Fenomena kontemporer** (Media sosial, Pinjol, Digital Economy)
- **Konsep kurikulum K13+** (Profil Pelajar Pancasila)
- **Sistem hint** untuk scaffolding kognitif

### 4. **Audio & Visual Gaming**
- BGM elektronik gamelan dari sumber publik
- Efek suara synthesized (click, success, fail)
- Gradasi warna emas (#ffd700) dengan tema gelap cyberpunk
- Glass morphism UI dengan Tailwind CSS

### 5. **Sistem Penilaian & Ranking**
- **+100 XP** — Jawaban benar (Nalar Kritis Luar Biasa)
- **-10 XP** — Menggunakan hint (penalti kecil)
- **Ranking Akhir**:
  - 🎓 PANCASILA SCHOLAR (≥90%)
  - 🧭 PENJELAJAH TANGGUH (<90%)

## 🚀 Cara Menggunakan

### **Desktop / Laptop**
1. Buka `index.html` di browser modern
2. Klik **"Mulai Ekspedisi"** di menu utama
3. Pilih karakter (Bima atau Srikandi)
4. Gunakan **WASD** untuk bergerak menuju node era
5. Ikuti dialog, baca dengan seksama, jawab quiz dengan analisis kritis

### **Mobile / Tablet**
1. Buka file di browser mobile
2. Virtual Joystick muncul otomatis di sudut kiri bawah
3. Gerakkan dengan menyentuh area joystick
4. Tap opsi jawaban dengan jari

### **Tombol Kontrol**
| Input | Fungsi |
|-------|--------|
| **W/↑** | Gerak ke atas |
| **A/←** | Gerak ke kiri |
| **S/↓** | Gerak ke bawah |
| **D/→** | Gerak ke kanan |
| **Sentuh opsi** | Pilih jawaban quiz |

## 🔊 Audio & SFX

### **Background Music**
- **Sumber**: Pixabay CCO (Royalty-free gamelan elektronik)
- **Volume**: 30% (tidak mengganggu pembelajaran)
- **Loop**: Otomatis, dapat diatur ulang

### **Sound Effects**
Sintetik menggunakan Web Audio API:
- 🔘 **Click** — Sine wave 600→1000Hz (0.1s)
- ✅ **Success** — Chord C#E (Triangle wave, 0.5s)
- ❌ **Fail** — Sawtooth descending 300→100Hz (0.4s)
